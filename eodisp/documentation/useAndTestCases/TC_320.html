Test Case TC_320

This page describes one of the test cases of the EODiSP.

This test case describes the steps required to generate an HLA wrapper skeleton using the HLA Wrapper Generator Application. The wrapper skeleton is generated for the Sim_Controller federate of the EarthCARE Simulator.

Contents

  • Overview
  • Steps

Overview

Identifier TC_320
Name Generate an HLA Wrapper Skeleton
Primary Actor ETH_2 Model Owner
Description Describes the steps to generate the HLA wrapper skeleton for the sim_controller federate of the EarthCARE Simulator using the HLA Wrapper Generator Application.
Pre-Condition (1) The SOM for the Sim_Controller federate is available. (2) The HLA Wrapper Generator Application is installed.
Post-Condition A set of Java files implementing the HLA wrapper skeletong for the Sim_Controller are available.

top ^

Steps

  1. Actor checks that the SOM complies with the the XML Schema available from the download section of this web page.
  2. Actor runs the HLA Wrapper Generator Application application.
