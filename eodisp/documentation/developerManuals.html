Developer Manuals

This page provides developer-level information about the EODiSP. The EODiSP is a framework that can be instantiated to create a simulator. The term 'developer' designates both persons who wish to instantiate the EODiSP framework, and persons who wish to modify or extend the EODiSP framework.

The EODiSP makes extensive use of free and open software from third parties. No attempt is made to provide documentation for these third-party applications although, where appropriate, pointers to where the documentation can be found are provided.
