Bug Tracking

Please use the bug tracker for bug reports and feature requests. Accessing the bug tracker needs a login and a password. They can be requested on the bug tracker website:

  • http://control.ee.ethz.ch/~ceg/mantis/

Note that the bug tracking site also hosts other projects. Please choose the EODiSP project in the upper right corner of the bug tracker website to file a bug in the right project (only available if logged in).
